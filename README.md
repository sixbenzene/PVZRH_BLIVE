### 日志
- 🔥2024/12/13:将代码整体上传到仓库🔥


### 如何开始
- 拉取代码
```
git clone https://github.com/sixbenzene/PVZRH_BLIVE.git
cd PVZRH_BLIVE
```
- 创建conda环境
```
conda create -n blive python=3.10
pip install -r requirements.txt
```
